# BotLanes

BotLanes is a Kanban board for managing [gstack](https://github.com/garrytan/gstack) agent tasks through a skill pipeline. Cards bind to durable OpenClaw sessions, support inline human replies, stage runs.

The board ships with gstack's skills as columns. It aids the visual flow in determining which card should move where, and the card provide feedback about what's occured and what to do next.

## How to Use BotLanes

1. Create cards (or tasks) in the Backlog you'd like an agent to handle
2. Drag the card into any columns to instantly kick off an agent session
3. Check in on card statuses and move them when a stage completes
4. OR (coming soon) drag the first card and let it run on auto-pilot, moving the card from one stage to the next as recommended by each skill
