# botlanes

botlanes is a Kanban board for managing [gstack](https://github.com/garrytan/gstack) agent tasks through a skill pipeline.

The board ships with gstack's skills as columns. It aids the visual flow in determining which card should move where, and the card provide feedback about what's occured and what to do next.

## How to use botlanes

1. Create a project by choosing a directory where you want your agents to work
1. Create cards (or tasks) in the Backlog you'd like an agent to handle
1. Drag the card into any columns to instantly kick off an agent session
1. Check in on card statuses and move them when a stage completes
